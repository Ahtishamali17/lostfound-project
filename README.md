# AI Lost & Found System

This project is an AI-powered Lost and Found web application designed to match lost and found items automatically.

## Features
- Lost item reporting
- Found item reporting
- NLP text similarity (TF-IDF)
- Image upload and comparison
- Match detection
- Admin dashboard
- Match history tracking

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- Python (NLP + Image similarity)
